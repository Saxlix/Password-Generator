Программа для создания надежных парольных фраз методом Diceware. Реализована на Python с использованием PyQt5. Это позволяет генерировать пароли, которые легко запомнить человеку, но крайне сложно подобрать автоматизированными системами перебора.ФункционалПриложение позволяет настраивать количество слов и ограничения по их длине. После генерации пароль отображается на экране и автоматически сохраняется в локальную базу данных SQLite. Предусмотрен экспорт истории всех созданных фраз в текстовый файл для дальнейшего анализа или хранения. Для удобства работы с паролем реализована визуализация в виде карточки, которая генерируется с помощью библиотеки Pillow.Технические особенности. Архитектура приложения построена на разделении слоев: пользовательский интерфейс, бизнес-логика и работа с хранилищем данных.  Использование менеджеров компоновки гарантирует корректное отображение элементов при изменении размеров окна.  Для взаимодействия с базой данных используется стандартный модуль sqlite3 с предварительной настройкой транзакций.  Работа с изображениями реализована через библиотеку Pillow с конвертацией в формат, поддерживаемый Qt.  Весь процесс работы программы сопровождается ведением лог-файла app.log для диагностики возможных ошибок
